# 📦 Logistics Performance Analysis using SQL & Power BI

## 📌 Project Overview

This project focuses on analyzing logistics and delivery performance using SQL and Power BI. The objective is to evaluate key performance indicators (KPIs), identify delivery inefficiencies, and generate actionable insights to improve operational efficiency.

---

## 🎯 Objectives

* Analyze delivery performance and delays
* Calculate key KPIs such as:

  * Average Delivery Delay
  * On-Time Delivery Percentage
  * Average Traffic Delay
* Identify inefficiencies in routes and operations
* Provide data-driven recommendations for improvement

---

## 🛠️ Tech Stack

* **SQL** – Data cleaning, transformation, and KPI calculation
* **Power BI** – Data visualization and dashboard creation
* **Excel / CSV** – Data source

---

## 📊 Key KPIs

* 📉 **Average Delivery Delay** (in days)
* ⏱️ **On-Time Delivery Percentage**
* 🚦 **Average Traffic Delay per Route**

---

## 📈 Dashboard Insights

* On-time delivery rate is relatively low (~56%), indicating performance gaps
* Average delivery delay exceeds 5 days
* Traffic congestion significantly impacts delivery efficiency
* Certain regions and routes show higher delays

---

## 🧠 Key Insights

* Delivery performance is suboptimal and requires improvement
* Traffic delay is a major contributing factor
* Some routes and regions consistently underperform

---

## 🚀 Recommendations

* Optimize high-delay routes
* Improve scheduling and delivery planning
* Monitor underperforming regions
* Use data-driven strategies for operational improvement
